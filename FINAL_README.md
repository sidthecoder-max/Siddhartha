# Artificial Intelligence Assignment

## Overview

This repository contains implementations of:

-   Turing Test Architecture Design
-   CAPTCHA System Architecture
-   Simple Reflex AQI Agent (Dataset-based)
-   Uninformed Search Algorithms (BFS & DFS)
-   Missionaries and Cannibals Problem

------------------------------------------------------------------------

## AQI Agent (Dataset-Based)

Reads environmental pollutant data (e.g., PM2.5) from CSV and computes
AQI using standard breakpoint formula.

Run:

    python aqi_agent.py

------------------------------------------------------------------------

## Search Algorithms

Includes:

-   Breadth First Search (BFS)
-   Depth First Search (DFS)
-   Missionaries & Cannibals Implementation

Run:

    python missionaries_cannibals.py
